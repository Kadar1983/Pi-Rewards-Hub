# Pi Rewards Hub

Testnet-ready Pi Browser app. Place `validation-key.txt` in `public/` and deploy to Vercel.

## Run locally
1. npm install
2. npm run dev
3. Open http://localhost:3000 inside Pi Browser (mobile).

## Deploy
- Push to GitHub and connect to Vercel.
- In Pi Developer, set your domain (subdomain) and verify; Pi will check `https://<your-domain>/validation-key.txt`.
